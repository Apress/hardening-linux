This Zip file contains the source and examples from the book, Hardening Linux.

Each directory contains source code and/or examples from particular chapters.


The examples contained are:

Chapter02

bastion.sh		A shell script to set your iptables rules for a bastion host.

Chapter05

log_testing.sh		A shell script for sending test syslog entries using logger.
syslog.sql		SQL commands to set up a MySQL database for syslog entries.
syslog2mysql.sh		A shell script to second syslog entries to a MySQL database.
syslog-ng.conf		A syslog-NG sample configuration file.

Chapter07

amavis_master.cf	Add Amavis support to Postfix master.cf file.
anti_spam_main.cf	A simple anti-spam configuration to add to the Postfix main.cf file.
clamav.conf		Clam-AV daemon configuration file.
freshclam.conf		Freshclam daemon configuration file.

Chapter09

cyrus_imap_init		An init script for Cyrus IMAP

Chapter10

anonymous_vsftpd.conf	A vsftpd.conf file for an anonymous FTP server.
local_user_vsftpd.conf	A vsftpd.conf file for an FTP server that allows local user access.

Chapter11

named_caching.conf	A named.conf file for a BIND Caching Server.
named_name_server.conf	A named.conf file for a BIND Name Server.
named_split_dns.conf	A named.conf file for split DNS BIND server.
named_init		An init script for BIND.
